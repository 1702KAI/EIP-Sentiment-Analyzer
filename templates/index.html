Three-Stage Analysis Pipeline

Stage 1: Sentiment Analysis
  • VADER sentiment scoring
  • EIP/ERC number extraction
  • Text preprocessing
  • Sentiment aggregation
Stage 2: Data Enrichment
  • EIPs Insight API integration
  • Metadata fetching
  • PR status tracking
  • Review data processing
Stage 3: Final Analysis
  • Data merging & deduplication
  • Status conflict detection
  • Summary statistics
  • Final output generation
CSV Requirements

Your CSV file must contain the following columns:

  • paragraphs - Text content from paragraphs
  • headings - Text content from headings
  • unordered_lists - Text content from lists
  • topic - Topic/title containing EIP or ERC identifiers
Tip: The analyzer will automatically extract EIP and ERC numbers from the topic field using pattern matching.
